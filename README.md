# 丹港 001

This repository contains a GitHub Pages-ready player for `丹港影片制作/001`.

## Files

- `index.html` - the web player page
- `assets/danhabor-001-cover.jpg` - poster image
- `assets/danhabor-001.mp4` - 720p web playback version

The original source video is about 1.1GB, which is too large for a normal GitHub repository file. The committed video is a compressed playback copy sized under GitHub's 100MB single-file limit.
